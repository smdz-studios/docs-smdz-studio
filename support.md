# Support

For official support with any **SMDZ Studios** FiveM script, please use one of the following channels:

- **Email:** `smdzstudios@gmail.com`  
- **Discord:** [Join the SMDZ Studios Discord](https://discord.gg/XraafphzrQ)

---

## Required information for support

To receive fast and accurate support, always provide:

1. **Script name**  
   - Example: `smdz_example`, `smdz_vehicleshop`, etc.

2. **Script version**  
   - Example: `v1.0.3` (you can usually find this in the config or the resource manifest).

3. **Framework and version**
   - ESX / QBCore / Standalone.
   - Include the framework version if relevant (for example “ESX Legacy”, specific QBCore commit, etc.).

4. **Tebex purchase ID**
   - The **order ID / transaction ID** from Tebex proving your purchase.

5. **Clear description of the issue**
   - What you expected to happen.
   - What actually happens instead.
   - Any console or F8 errors.

6. **Reproduction steps**
   - Short, numbered list so we can reproduce the problem on a test server.

- Any relevant configuration snippets (sanitized, without secrets).
- Exact steps so we can reproduce the issue on a clean test server.
