# Scripts Overview

This section lists all **SMDZ Studios** scripts that have dedicated documentation pages.

> When you add a new script, follow these steps:
> 1. Create a new Markdown file in the `scripts` folder using `example-script.md` as a template.
> 2. Update this list to include a link to the new script.
> 3. Update `/_sidebar.md` if you want quick access from the main menu.
